# Дорожная карта Project TODO

Живой план разработки local-first приложения для заметок, задач и управления выбранными файлами.

Обозначения:

- `[x]` — реализовано и доступно в текущем прототипе.
- `[ ]` — ещё не реализовано или требует доведения до рабочего состояния.
- **MVP** — первая Windows-версия, которой можно ежедневно пользоваться без риска потерять данные.
- **После MVP** — полезные расширения, которые не должны задерживать первый стабильный релиз.

## Цель продукта

Project TODO объединяет три повседневных сценария в одном локальном рабочем пространстве:

1. Быстро записать и структурировать мысль.
2. Превратить мысль в задачу и довести её до завершения.
3. Собрать связанные файлы в понятные коллекции без необходимости переносить весь компьютер внутрь приложения.

Основные принципы:

- данные принадлежат пользователю и доступны без интернета;
- интерфейс быстрый, спокойный и структурный, с визуальным направлением в духе Linear;
- приложение не требует платного сервера для базовой работы;
- заметки, задачи и файлы связаны внутри одного рабочего пространства;
- Windows является первой целевой платформой.

## Критерии готовности MVP

MVP считается готовым, когда выполнены все условия:

- [ ] Пользователь может установить приложение на Windows без среды разработки.
- [ ] После перезапуска сохраняются workspace, заметки, задачи, файлы и настройки.
- [ ] Все основные операции работают без подключения к интернету.
- [ ] Пользователь может создать, отредактировать, найти, удалить и восстановить заметку.
- [ ] Пользователь может создать задачу, назначить срок и статус, затем завершить её.
- [ ] Пользователь может импортировать файл, найти его и открыть оригинал.
- [ ] Экспорт и восстановление workspace проверены на реальных данных.
- [ ] Аварийное закрытие не приводит к потере ранее сохранённых данных.
- [ ] Нет известных блокирующих ошибок в критических сценариях.
- [ ] Автообновление успешно проверено минимум между двумя тестовыми релизами.
